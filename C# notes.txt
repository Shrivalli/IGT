Execution in .NET core

1. C# source code - C# compiler - convert the C# source code into MSIL (Microsoft Intermediate Language)/CIL(Common IL)
2. VB Source code - VB Compiler - convert the VB source code into MSIL

MSIL --> CLR which has JIT compiler --> Native code which OS can execute

Source code --> compilers --> MSIL --> CLR(JIT) --> Native code (exe)
